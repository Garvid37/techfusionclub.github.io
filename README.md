## Requirements

- GNU/Linux, Unix, Windows or macOS
- Ruby version 2.1 or above, including all development headers
- RubyGems
- Node.js
- npm || yarn
- gulp

### Clone this repo

### Install Jekyll with RubyGems
`$ gem install jekyll`

### Install dependencies
`$ npm install` or `$ yarn install`

### Serve with gulp
`$ gulp`
